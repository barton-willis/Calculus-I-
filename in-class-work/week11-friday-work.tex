 \question Given these facts about a function $J$, draw a sketch of $J$ that has all the given properties:
 
  \begin{tabular}{|c|c|}  \hline \hline 
 $ x < 2$  & $J^\prime(x) < 0$ \\  \hline
 $ 2 < x < 3$   & $J^\prime (x) = 0$ \\  \hline
  $ 3 < x $   & $J^\prime(x) > 0$  \\ \hline
 \end{tabular}
 
 Also $J(2) = 5$.
